# Portfolio Website

A modern, dark-themed portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- 🎨 Dark minimalist design
- 📱 Fully responsive and mobile-friendly
- ✨ Smooth animations with Framer Motion
- 🚀 Built with Next.js 14 and TypeScript
- 🎯 Modern UI/UX with Tailwind CSS

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app router pages
- `/components` - Reusable React components
- `/data` - Data files (projects, etc.)
- `/public` - Static assets

## Pages

- **Home** (`/`) - Landing page with personal info and social links
- **Projects** (`/projects`) - Showcase of portfolio projects
- **Skills / Studies** (`/skills`) - Technical skills and education
- **About Me** (`/about`) - Personal information
- **Contact** (`/contact`) - Creative contact form and social links

## Customization

Update the following files to customize the portfolio:

- `app/page.tsx` - Update personal information
- `data/projects.ts` - Add/modify projects
- `app/skills/page.tsx` - Update skills and education
- `app/about/page.tsx` - Update about section
- `app/contact/page.tsx` - Update contact information

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
